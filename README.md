##  Getting Started

To run Echo-OS on your local machine, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/LuisZavoianu/Echo-OS.git](https://github.com/LuisZavoianu/Echo-OS.git)
   cd Echo-OS

   Install dependencies: npm install
   Start the development server: npm run dev
   Build for production: npm run build

   Project Structure
src/kernel/ - The central logic and OS engine.

src/components/nodes/ - Visual definitions for various node types.

src/hooks/ - Custom hooks for state management and node connectivity.

src/assets/ - Static assets and global styling.
